# ⚡ QR Forge — Instant QR Code Generator

> A premium, free QR code generator that works entirely in your browser — no signup, no data stored.

🔗 **Live App:** [https://your-username.github.io/qr-forge](https://your-username.github.io/qr-forge)

---

## ✨ Features

- **8 QR Code Types** — Website, Video, Payment (UPI), Text, Email, Phone, Wi-Fi, Contact (vCard)
- **Full Customization** — Foreground/background colors, 8 presets, size slider (150–600px)
- **Error Correction** — L / M / Q / H levels
- **Download Options** — PNG, SVG, or copy to clipboard
- **100% Private** — All QR codes are generated locally in your browser
- **Premium Design** — Dark glassmorphism, animated gradients, smooth animations

## 🚀 Use Cases

| Type | Use Case |
|------|----------|
| 🌐 Website | Share any URL instantly |
| ▶️ Video | Link to YouTube / Vimeo |
| 💳 Payment | Accept UPI payments |
| 📶 Wi-Fi | Guest network access |
| 👤 Contact | Share vCard / business info |
| 📧 Email | Pre-filled email links |
| 📞 Phone | Direct call links |
| 📝 Text | Plain messages |

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **Vanilla CSS** — Glassmorphism design system
- **Vanilla JS** — Zero framework, fast load
- **[qrcodejs](https://github.com/davidshimjs/qrcodejs)** — QR generation library

## 🏃 Run Locally

Just open `index.html` in any browser — no build step required!

```bash
# Or serve with Python
python3 -m http.server 8080
# Then visit http://localhost:8080
```
